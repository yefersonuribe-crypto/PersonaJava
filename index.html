Audio y Video Nativo

Adiós Flash Player. HTML5 lo hace nativo.

  • controls: Muestra play/pausa.
  • loop: Se repite infinitamente.
  • autoplay + muted: El video arranca solo (pero sin sonido para no molestar).
<video controls muted autoplay>
  <source src="intro.mp4">
</video>

<audio controls>
  <source src="bgm.mp3">
</audio>

CSS para Multimedia (Responsive)

Problema: Las imágenes grandes rompen el diseño en celulares.

Solución: La regla de oro del frontend:

img, video, iframe {
  max-width: 100%;
  height: auto;
  display: block;
}
